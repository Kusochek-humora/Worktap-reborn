Установка
=========

Необходим nodejs >= 13

```shell
npm install
npm install --global gulp-cli
```

Запуск
======

Пересборка всего:

```shell
gulp
```

Пересборка всего и запуск сервера:

```shell
gulp dev
```

Сервер должен быть на 127.0.0.1:3000
